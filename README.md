# userSignIn

### Install package manager

```shell
$ npm init
```

### Install nodemon

```shell
$ npm install --save-dev nodemon
$ npm run dev
```

---

### Heroku initial Deploy

```shell
$ heroku login
$ heroku create
$ heroku git:remote -a thawing-falls-30759
$ git push heroku master
$ heroku open
```

### Deploy changes to Heroku

```shell
$ git add .
$ git commit -am "make it better"
$ git push heroku master
```
